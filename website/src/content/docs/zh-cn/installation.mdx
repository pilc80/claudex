## 安装方式

<Tabs>
  <TabItem label="一键安装" icon="terminal">
    ```bash
    curl -fsSL https://raw.githubusercontent.com/pilc80/claudex/main/install.sh | bash
    ```
    安装脚本会自动检测你的操作系统和架构，从 GitHub 下载最新版本，并安装到 `~/.local/bin/claudex`。

    自定义安装目录：
    ```bash
    CLAUDEX_INSTALL_DIR=/usr/local/bin curl -fsSL https://raw.githubusercontent.com/pilc80/claudex/main/install.sh | bash
    ```
  </TabItem>
  <TabItem label="Cargo" icon="seti:rust">
    ```bash
    cargo install --git https://github.com/pilc80/claudex
    ```
    需要 Rust 工具链。二进制文件将安装到 `~/.cargo/bin/claudex`。
  </TabItem>
  <TabItem label="GitHub Releases" icon="github">
    从 [GitHub Releases](https://github.com/pilc80/claudex/releases) 下载适合你平台的预编译二进制文件。

    可用目标：
    - `x86_64-unknown-linux-gnu`
    - `x86_64-unknown-linux-musl`
    - `aarch64-unknown-linux-gnu`
    - `aarch64-unknown-linux-musl`
    - `x86_64-apple-darwin`
    - `aarch64-apple-darwin`
    - `x86_64-pc-windows-msvc`

    解压并将二进制文件放入你的 `PATH`：
    ```bash
    tar xzf claudex-v*.tar.gz
    chmod +x claudex
    mv claudex ~/.local/bin/
    ```
  </TabItem>
</Tabs>

## 系统要求

| 要求 | 详情 |
|------|------|
| **操作系统** | macOS (Intel / Apple Silicon)、Linux (x86_64 / ARM64)、Windows (x86_64) |
| **Claude Code** | 必须已安装，且 `claude` 命令在 PATH 中可用 |
| **可选** | [Ollama](https://ollama.com) 用于智能路由和上下文引擎功能 |

## 验证安装

```bash
claudex-config --version
```
